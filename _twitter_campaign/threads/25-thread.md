TWEET 2/6:
Mobile AI apps: ChatGPT, Claude, Gemini all have mobile versions. Useful for on-call documentation or hospital rounds.

---

TWEET 3/6:
Security: Require password/biometric lock. Auto-lock after 1 minute. Encrypt device. These aren't optional.

---

TWEET 4/6:
Network safety: Use hospital WiFi or VPN, not public networks. Coffee shop WiFi + patient data = violation waiting to happen.

---

TWEET 5/6:
Screen privacy: Privacy screens prevent over-shoulder viewing. Important in shared spaces like nursing stations.

---

TWEET 6/6:
Lost device risk: Remote wipe capability is essential. If device is lost, patient data must be erasable remotely.
